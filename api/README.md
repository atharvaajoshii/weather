# Weather App

A simple weather application built with Node.js, Express, and EJS that allows users to search for any city and view its current weather conditions in real time.

## Live Demo

https://weather-k148bfnv6-atharvaajoshiis-projects.vercel.app/

## Features

* Search weather by city name
* Real-time weather information
* Temperature display in Celsius
* Humidity information
* Weather condition icons
* Error handling for invalid city names

## Tech Stack

* Node.js
* Express.js
* EJS
* Axios
* OpenWeatherMap API

## How It Works

1. Enter a city name.
2. The application sends a request to the OpenWeatherMap API.
3. Weather data is fetched and processed on the server.
4. Current weather details are displayed to the user.
